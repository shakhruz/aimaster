# AI Master Course

> Запусти своего персонального AI-агента. Стань мастером, экспертом, лидером.

🔗 Live: https://aimaster.vercel.app

## Three Levels

- 🎓 **Master** — Personal AI agent for yourself
- 💼 **Expert** — Train others, consult, customize  
- 🚀 **Leader** — Lead AI transformation in organizations

## Stack
- Next.js 15 + TypeScript
- Tailwind CSS 4
- localStorage for checklist state
- i18n: Russian + English

## Run locally
```bash
npm install
npm run dev
```
